# Usage Example

## Stylus

set `--include ./node_modules/@noliaki/css-cubic-bezier-variable/stylus/` option

```sh
$ yarn build:stylus
# or
$ npm run build:stylus
```

## Scss

set `--include-path ./node_modules/@noliaki/css-cubic-bezier-variable/scss/` option

```sh
$ yarn build:scss
# or
$ npm run build:scss
```
